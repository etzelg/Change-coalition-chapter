# Israeli Political Events Dataset (March 2021 - July 2022)

## Overview
This dataset documents significant political events in Israel from March 2021 to July 2022. Events are selected based on their impact on public and party political discourse.

## Categories
Events are classified into the following categories:
- **Coalition**: Government formation, coalition agreements, coalition changes, party splits
- **Legislation**: Major laws passed, parliamentary votes, legislative initiatives
- **Terror Attacks**: Attacks against Israeli targets
- **Media Issues**: Major controversies, scandals, significant media coverage events
- **COVID-19**: Pandemic-related policies, lockdowns, vaccination campaigns
- **Economic Issues**: Economic policies, inflation, unemployment, budget concerns
- **Crises**: National emergencies, security incidents, protests

## Data Structure
Each month's file (`MMM-YYYY.csv`) contains:
- **Date**: Event date (DD/MM/YYYY format)
- **Event**: Brief event title
- **Category**: Category classification
- **Description**: Detailed description of the event
- **URL**: Source URL for verification

## Selection Criteria
- **Maximum 2-3 events per week**: Only the most significant events are included
- **Political Impact**: Events that significantly affected public and party political discourse
- **Verifiability**: All events are sourced from reliable news outlets and official sources

## Character Encoding
All CSV files are encoded in UTF-8 to properly support Hebrew text.

## Updates
Files are committed and pushed after completion of each month's data.
